## 初始化工作区
`git init`

## 全局配置
`git config --global user.name 'xxxx'`
`git config --global user.email 'xxxxx@xx.com'`

## 查看状态
`git status`

## 将工作区指定的内容添加到暂存区
`git add 文件夹或文件`

## 将暂存区的内容提交到本地仓库，生成版本
`git commit -m '备注信息'`

## 查看详细日志
`git log`

## 查看精简日志
`git log --oneline`
